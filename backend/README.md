## 项目说明

此项目基于 GIN、GORM 开发的 Web 后端项目，实现以下功能：
- 基于 Casbin 的权限管理
- 基于 WebSocket 的消息通知（聊天室）
- 基于 JWT 的用户登录鉴权


## 如何启动项目

开发环境： `cd backend ; go run main.go `
